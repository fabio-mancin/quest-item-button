# QuestItemButton

A one-click button for **usable quest items**, shown only when you're carrying
the item *and* standing in the zone where it's meant to be used — Retail-style,
for WoW Classic Anniversary (TBC 2.5.6 / Progression).

**Example:** quest *Conjurer Luminrath* in your log + *Luminrath's Mantle* in
your bags + you're in Netherstorm → the Mantle shows up as a clickable button.
Leave the zone or drop the quest and it disappears.

## Features

- Automatic show/hide driven by bags, quest log, and current zone
- Secure button — safe to click in combat
- Configurable trigger list, per-quest disables, and button position
- Settings persist across sessions (`/qib` or the options panel)

## Installation

Install via the CurseForge app, or drop the `QuestItemButton` folder into
`World of Warcraft/_anniversary_/Interface/AddOns/` and `/reload`.
